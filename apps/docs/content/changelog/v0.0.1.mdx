# v0.0.1 Release Notes

*Released: June 2026*

Version `0.0.1` marks the initial public alpha release of PlayerKit!

---

### Initial Features
* Unified `<Player />` component supporting HLS, YouTube, and MP4.
* Headless hooks: `useHlsPlayer()`, `useYoutubePlayer()`, and `useMp4Player()`.
* CSS variable-based styling via `@playerkit/ui`.
* Standard media controls: Play, pause, seek, volume, and fullscreen.

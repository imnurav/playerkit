# v0.0.2 Release Notes

*Released: Late June 2026*

Version `0.0.2` brings a major overhaul to the live streaming engine and improves error handling.

---

### Key Changes
* **Live Engine Overhaul** — Refactored the core handling for live broadcast DVR windows and edge detection.
* **Error Handling** — Better recovery from network failures and corrupted segments.
* **Layout Fixes** — Fixed visual layout bugs in the UI components for embedded players.

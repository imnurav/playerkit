# v0.0.4 Release Notes

*Released: Previous version*

Version 0.0.4 was a major architectural milestone for PlayerKit, introducing the unified orchestrator and enterprise token architecture.

## Highlights

* **Added:** The unified `<Player>` orchestrator component that auto-detects HLS, YouTube, and MP4 sources from the URL string.
* **Added:** `tokenFetcher` support for protected streams (HLS and MP4).
* **Added:** Query-parameter signing via `xhrSetup` for advanced CDN token architectures.
* **Added:** Mobile touch gestures (double-tap to seek, center-tap to pause).
* **Added:** The enterprise security shield (`disableDevOptions`).
